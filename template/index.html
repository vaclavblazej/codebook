Template

Template is part of code that is included at the beginning of all written solutions. The following is the general template used in this book. As taste is coder-specific we opt to use a minimal template.

template.h,template

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
typedef double ld;

You may want to create your own template to fit your needs and style. Consider using snippets and autocompletion instead of a complex template if you code on a machine you can configure.

The following commands are some of the more popular ones.
